## Apresentação

O **Thatent Editor** é um editor de documentos open-source baseado em Vue3 e Tiptap3, oferecendo uma experiência de edição online fluida, pensada para ser integrada ao ecossistema Thatent. Ele provê capacidades robustas de edição, ferramentas de autoria e uma interface intuitiva, suportando modo de paginação, sintaxe Markdown, edição de rich text, inserção de diversos tipos de nós, estilos de página, exportação e impressão de documentos.

Além disso, o Thatent Editor suporta extensões customizadas, internacionalização e temas (incluindo dark mode). Também oferecemos o componente de visualização [Thatent Viewer](https://www.google.com/search?q=https://github.com/devwarly/Thatent-Viewer), que você pode integrar em seus projetos para pré-visualizar documentos.

O principal diferencial do Thatent Editor é o código totalmente aberto e autônomo. Ele suporta deploy privado, permitindo que você utilize a ferramenta em ambientes de intranet sem preocupações com segurança de dados.

## Introduction

**Thatent Editor** is an open-source document editor based on Vue3 and Tiptap3, providing an online editing experience. It provides powerful document editing capabilities. Thatent Editor supports pagination and normal web layout, Markdown syntax, rich text editing, insertion of various node types, page style settings, document export, and printing.

Additionally, Thatent Editor supports custom extensions, multi-language settings, and dark theme. We also provide a standalone text preview component, [Thatent Viewer](https://www.google.com/search?q=https://github.com/devwarly/Thatent-Viewer), which users can integrate into their projects to preview documents.

Thatent Editor's largest feature is that its code is completely open source and controllable. It supports private deployment, allowing you to use it in an intranet environment without worrying about data security issues.

As an independent Vue3 plugin, Thatent Editor can be easily integrated into various Vue3 projects.

## Thatent Editor Mobile

O **Thatent Editor Mobile** é um editor de documentos para dispositivos móveis construído com Vue3 e Tiptap3. Ele suporta todo o fluxo "visualizar + editar + salvar", podendo ser integrado diretamente em sistemas de negócio como um editor independente. Redesenhado para cenários de toque, ele possui sua própria máquina de estados, salvamento automático, histórico, internacionalização e temas.

## Filosofia de Design

O Thatent Editor visa simplificar a edição de documentos em aplicações web, provendo capacidades de edição poderosas enquanto mantém a conveniência de aplicações web modernas. Seja para sistemas de gestão, escrita acadêmica, colaboração de documentos em equipe ou organização de notas pessoais, o Thatent Editor é seu assistente confiável.
